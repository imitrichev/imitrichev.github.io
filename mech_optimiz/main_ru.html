Что такое mech_optimiz?

Логотип проекта

mech_optimiz - это программа для оценки кинетических параметров микрокинетических механизмов реакций в каталитических процессах.

Оно может быть использовано как учеными для построения и проверки кинетических моделей, так и инженерами для создания детальных и сокращенных кинетических моделей, которые могут быть использованы в современных CFD-кодах (FLUENT, COMSOL, Star-CCM+).

Отличительные особенности ПО:

  • Использование параллельных вычислений и эволюционного алгоритма для оценки кинетических параметров
  • Последовательности Соболя для генерирования случайных точек в процессе поиска
  • Анализ чувствительности/степени управления скоростью
  • Термодинамическая непротиворечивость кинетических параметров
  • Возможна работа с mongodb с использованием база данных кинетических параметров
  • Построение доверительных интервалов
  • Набор встроенных полезных инструментов
