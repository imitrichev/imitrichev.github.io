Логотип проекта

Что такое mech_optimiz? - это программное обеспечение для оценки кинетических параметров микрокинетических механизмов реакций в каталитических процессах.

Оно может быть использовано как учеными для построения и проверки кинетических моделей, так и инженерами для создания детальных и сокращенных кинетических моделей, которые могут быть использованы в современных CFD-кодах (FLUENT(R), COMSOL(R), Star-CCM+(R)).

Отличительные особенности ПО:

  • Использование параллельных вычислений и эволюционного алгоритма для оценки кинетических параметров
  • Последовательности Соболя для генерирования случайных точек в процессе поиска
  • Анализ чувствительности/степени управления скоростью
  • Термодинамическая непротиворечивость кинетических параметров
  • Возможна работа с mongodb с использованием базы данных кинетических параметров
  • Построение доверительных интервалов
  • Набор встроенных полезных инструментов

Кинетические механизмы, полученные с помощью mech_optimiz

Работа в области создания микрокинетических механизмов реакций вдохновлена чтением публикаций проф. O. Deutschmann и его сайтом с кинетическими механизмами. Все кинетические механизмы (формат файлов SURFACE CHEMKIN(R) .inp), представленные на странице, были тестированы на работоспособность в mech_optimiz и являются термодинамически непротиворечивыми для диапазона условий применимости.
Описание алгоритма термодинамического согласования кинетических параметров представлено в публикации
