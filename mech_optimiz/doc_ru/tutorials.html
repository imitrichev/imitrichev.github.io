Обучающий пример - 1. Решение прямой кинетической задачи для системы NO+CO/Pt

Скопируйте mech_optimiz и все файлы, прилагаемые в директории seed в отдельную директорию. Если вы получали исходный код программы, директория seed находится внутри ветки кода master.

Запустите программу с конфигурационным файлом noco_FKP.txt и файлом сохранения noco_FKP.txt-1.

./mech_optimiz noco_FKP.txt noco_FKP.txt-1

Программа должна просчитать одну итерацию ГА, оценивая приспособленность только одной особи, помеченной "-1" в файле сохранения. Так как это особь имеет фенотип, совпадающий с параметрами кинетического механизма mech.inp, то будет оценена целевая функция для данного кинетического механизма.

Программа должна завершиться без ошибок с выводом на экран сообщения:

Program finished with result:
Solution was found at generation 1 with fitness 588.596 with MAX_GENERATIONS = 1 reached

Часть ошибок и дополнительных сведений пишется в лог-файл error.log. В данном случае файл не должен содержать ошибок (Error). Файл, скорее всего, будет оканчиваться строками:

Total deltaH LSR
3.49997e+07
Total deltaS LSR
52.7589
Total deltaG LSR
5.27788e+07
Total deltaHandS LSR
4.51024e+07

с оценкой термодинамической непротиворечивости кинетического механизма.
