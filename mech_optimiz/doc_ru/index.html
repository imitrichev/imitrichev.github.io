Пояснение, как использовать документацию (и mech_optimiz в целом)

Attention! The source code and installation instructions are not distributed at this moment. It is by default limited by institutional usage. If you want to use it, please, contact imitrichev at muctr (RU).

Здравствуйте!

Начинать советую с чтения раздела Идея и общая логика программы , чтобы понять, подойдет ли эта программа Вам. Программа работает в настоящее время только на ОС Linux.

Возможно, далее полезно будет ознакомиться с обучающими примерами Обучающие примеры для mech_optimiz, чтобы понять лучше, как работает программа, и что вы точно сможете с ее помощью получить (без изменения кода).

Выполните шаг 1 руководства по установке (installation ) для установки необходимых библиотек.
Также установите Cantera по официальному руководству, или выполните шаг 2 руководства, указанного выше (для компиляции Cantera из исходных файлов).

Если для вашей системы не подойдет скомпилированная бинарная версия mech_optimiz, то нужно будет скомпилировать программу на целевой машине (можно поместить исполняемый файл на распределенные файловые системы, например, NFS). Для этого нужно пройти от начала до конца руководство по установке. installation

Если вы хотите использовать возможность подключения базы данных с кинетическими параметрами - то нужно также пройти шаги по установке/компиляции требуемых пакетов и API (dbinstallation). Описание формата файла для базы данных дано в разделе Работа с базой данных реакций.

Синтаксис входных файлов (и, частично - выходных) описан в разделе: Использование mech_optimiz

Если вы хотите познакомиться подробнее с математической моделью, которую использует mech_optimiz, см. страницу Устройства решателей в mech_optimiz

Если вам нужна информация о дополнительных инструментах и возможностях mech_optimiz - посмотрите специальные страницы

  • Кластеризация данных
  • Анализ данных с помощью метода главных компонент
  • Учет латеральных взаимодействий в mech_optimiz
  • Частицы, занимающие несколько активных центров
  • Термодинамическая непротиворечивость кинетических механизмов
