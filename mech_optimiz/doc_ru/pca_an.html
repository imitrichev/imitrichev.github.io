Анализ данных с помощью метода главных компонент

На данной странице обсуждается реализация МГК в mech_optimiz.

Метод главных компонент (МГК, англ. Principal component analysis, PCA) - одно из средств анализа данных, позволяющих выявить скрытые закономерности в данных.

Суть метода главных компонент заключается в понижении размерности данных (Метод Главных Компонент (PCA), © 2008 Алексей Померанцев, Российское хемометрическое общество), переходе к системе координат меньшей размерности. Каждая новый координатный вектор (главная компонента) - есть линейная комбинация старых координатных векторов, то есть параметров. Коэффициенты линейного разложение главной компоненты по базису старых координатных векторов называются нагрузками. Все точки набора данных называют образцами. Координаты всех образцов в новом базисе называются счетами.

Мы использовали NIPALS-алгоритм [14] вычисления главных компонент, который вычисляет их итерационно, одну за другой, начиная от самых важных (от первой), объясняющих наибольшую часть остаточной дисперсии.
