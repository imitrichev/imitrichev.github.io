What is mech_optimiz?

Project logo

mech_optimiz is a software for parameter estimation in microkinetic reaction mechanisms of catalytic processes.

It could be used by scientists to build and verify kinetic models as well as by engineers to create detailed and lumped kinetic models that could be incorporated into modern CFD codes (FLUENT, COMSOL, Star-CCM+).

The distinctive features are:

  • Use of parallel calculations and evolutionary method for parameter estimation
  • Sobol sequence used in generating random search points
  • Sensitivity/rate control analysis
  • Thermodynamic consistency of parameter set
  • Works with mongodb, can use database for kinetic parameters
  • Construction of confidence intervals
  • A set of inbuilt useful tools
